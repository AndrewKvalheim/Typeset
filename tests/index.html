Typeset.js

A library for processing HTML to make typesetting straightforward. It's designed to run on the server and requires no client-side JavaScript

Accepts a HTML string and returns the same HTML with:

  • Hanging punctuation
  • Hyphen substitution
  • Small-caps for acronyms
  • Elipses substitution
  • Quote substitution
  • Hair spaces between ratios (2:1)
  • Encode hyphenation breakpoints?
  • Range dash for ranges (1990-2000)
