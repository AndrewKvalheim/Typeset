PREPRESS.js

An HTML pre-processor for better typography. It requires no client-side JavaScript and hardly any CSS.

Accepts an HTML string and returns the same HTML with:

  • Hanging punctuation
  • Soft hyphen insertion
  • Small-caps conversion
  • Elipses substitution
  • Quote substitution
  • Hair spaces between ratios
  • Range dash substitution
  • Hyphen substitution

How it works?

Typeset is available on npm:

npm install typeset

To process some HTML:


var typeset = require('typeset');
var html = '<p>"Hello there," said the fox's wife.</p>';
var processedHTML = typeset(html);

You can also pass an options object:

typeset(html, {disable: ['smallCaps'], language: 'English'});
