Typeset.js

An HTML pre-processor for web typography. It uses no client-side JavaScript but gives you:

  • Hanging punctuation
  • Soft hyphen insertion
  • Optical margin outdents
  • Small-caps conversion
  • Hair spaces between ratios
  • Range dash substitution
  • Hyphen substitution
  • Quote substitution
  • Elipses substitution

How to get it

npm install typeset

How to use Typeset.js

var typeset = require('typeset');
var html = '<p>"Hello," said the fox.</p>';
var output = typeset(html);

Then modify typeset.css (<1kb) to match your font and include it on your website. Learn more about the options on Github.
