How to get it

npm install typeset

The code is also available on on Github.

How to use it

var typeset = require('typeset');
var html = '<p>"Hello," said the fox.</p>';
var output = typeset(html);

Modify typeset.css (<1kb) to match the metrics of your font and include it on your page. Learn more about the options on Github.

About

This page is set in Charter, a libre typeface. This project started as a collection of libraries used by Blot for typesetting blog posts. This would not be possible without the work of Bram Stein and Dr. Drang.
