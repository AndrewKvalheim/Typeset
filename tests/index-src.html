TYPESET.js

An HTML pre-processor for web typography. It uses no client-side JavaScript but gives you:

  • Hanging punctuation
  • Soft hyphen insertion
  • Optical margin outdents
  • Small-caps conversion
  • Hair spaces between ratios
  • Range dash substitution
  • Hyphen substitution
  • Quote substitution
  • Elipses substitution

How to get

npm install typeset

How to use

var typeset = require('typeset');
var html = '<p>"Hello there," said the fox's wife.</p>';
var output = typeset(html);

Then modify typeset.css (1kb) to match your font and include on your website.

Options

typeset(html, {disable: ['smallCaps'], language: 'English'});
