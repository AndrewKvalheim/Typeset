Type­set.js

A li­brary for pro­cess­ing HTML to make type­set­ting straight­for­ward. It’s de­signed to run on the server and re­quires no client-side JavaScript

Ac­cepts a HTML string and re­turns the same HTML with:

  • Hang­ing punc­tu­a­tion
  • Hy­phen sub­sti­tu­tion
  • Small-caps for acronyms
  • Elipses sub­sti­tu­tion
  • Quote sub­sti­tu­tion
  • Hair spaces be­tween ra­tios (2:1)
  • En­code hy­phen­ation break­points?
  • Re­quires a small CSS file and no client-side JS.
  • Range dash for ranges (1990-2000)
