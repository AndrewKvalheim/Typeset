Typeset.js

A typographic pre-processor for your website's HTML which offers:

  • Real hanging punctuation
  • Soft hyphen insertion
  • Optical margin alignment
  • Small caps detection
  • Punctuation substitution
Get the code on GitHub > 
>  npm install typeset

How to use it

var typeset = require('typeset');
var html = '<p>"Hello World"</p>';
var output = typeset(html);

The library does not require any client-side JavaScript. You can run it on the client if circumstances dictate but try to avoid it if you can.

Hanging punctuation and optical margin alignment features require a few lines of CSS, which you'll need to modify based on the metrics of the typeface you use.

Modify typeset.css (1k) to match the metrics of your font and include it on your page. Learn more about the options on Github.
