Type­set.js

An HTML pre-proces­sor forweb ty­pog­ra­phy. It uses noclient-side JavaScript but gives you:

  • Hang­ing punc­tu­a­tion
  • Soft hy­phen in­ser­tion
  • Op­ti­cal mar­ginout­dents
  • Small-capscon­ver­sion
  • Hair spaces be­tween ra­tios
  • Range dash sub­sti­tu­tion
  • Hy­phen sub­sti­tu­tion
  • Quote sub­sti­tu­tion
  • Elipses sub­sti­tu­tion

How to get it

npm install typeset

How to useType­set.js

var typeset = require('typeset');
var html = '<p>"Hello," said the fox.</p>';
var output = typeset(html);

Then mod­ify type­set.css (<1kb) to match your font and in­clude iton yourweb­site. Learn more about theop­tions on Github.
